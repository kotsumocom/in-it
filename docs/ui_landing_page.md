# ランディングページ（トップ）UI 仕様書

## 概要

**メンター（講師）登録促進**に特化したランディングページ。  
生徒向け導線はヘッダーの「メンター一覧」リンクのみ。

---

## ターゲット

| 項目       | 内容                                           |
| ---------- | ---------------------------------------------- |
| プライマリ | **メンター（講師）候補**                       |
| ペルソナ   | 既存プラットフォームの高手数料に不満を持つ講師 |
| 目的       | メンター登録への誘導                           |
