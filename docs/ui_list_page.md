# 講師一覧ページ UI仕様書

## 概要

講師を「発見」するためのメインページ。
シングルカラム、カード型レイアウト。

---

## デザイン方針

| 項目 | 内容 |
|------|------|
| レイアウト | シングルカラム |
| カード | 角ばったエッジ（Firebase風） |
| カラー | Blue #1976D2 / Teal #00897B |

---

## ページ構成（ワイヤー）

### 1. ヘッダー
- ロゴ（イニット）
- 検索バー（キーワード）
- ログイン/登録ボタン

### 2. フィルターセクション
- カテゴリ選択（チップ形式、横スクロール）
- 例: `すべて` `プログラミング` `語学` `ビジネス` `クリエイティブ`
- 選択中は塗りつぶし、未選択はアウトライン

### 3. 講師カード一覧
- 縦スクロール（無限スクロール or ページネーション）
- 各カード構成:

```
┌────────────────────────────────┐
│ [アイコン]  名前 ✓本人確認     │
│             タグライン          │
│ ─────────────────────────────  │
│ [タグ] [タグ] [タグ]           │
│                                │
│ [プロフィールを見る →]         │
└────────────────────────────────┘
```

### 4. カード内要素

| 要素 | 仕様 |
|------|------|
| アイコン | 48px、円形 or 角丸正方形 |
| 名前 | 太字、#212121 |
| バッジ | 本人確認済みなら表示 |
| タグライン | 1行、グレー |
| タグ | 最大3つ、ピル型、単一カラー |
| CTA | テキストリンク or ゴーストボタン |

### 5. 空状態
- 検索結果0件時のイラスト + メッセージ
- 「条件を変えて検索してください」

### 6. ローディング
- スケルトンカード表示

---

## 検索・フィルター仕様

### キーワード検索
- 名前、タグライン、カテゴリを対象
- 部分一致
- デバウンス（300ms）

### カテゴリフィルター
- 単一選択 or 複数選択（TBD）
- URLクエリパラメータに反映（共有可能）

## ソート（MVP後）

- 新着順（デフォルト）
- レビュー評価順（Phase 2）

## レスポンシブ

| 画面幅 | カード幅 |
|--------|----------|
| モバイル | 100% |
