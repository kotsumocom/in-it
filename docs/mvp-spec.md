# **メンタープラットフォーム: MVP Specification**

## **1. MVP の目的 (Goal)**

**検証仮説:**
「手数料ゼロ・低掲載料（月額 1,000 円/スペース）のメンターマーケットプレイスは、高手数料に不満を持つ講師層を惹きつけ、持続可能なビジネスとして成立するか？」

**ビジネスモデル:**

- 月額掲載料: **1,000 円（税別）/ スペース**
- 年額掲載料: **10,000 円（税別）/ スペース**（2 ヶ月分お得）
- 取引手数料: **0%**（直接取引 OK・外部リンク自由）
- 決済: Stripe Subscription

---

## **2. ターゲットユーザー**

### 講師（メンター）側

- 既存プラットフォーム（Menta, ココナラ, ストアカ等）の高手数料（20-30%）に不満を持つ層
- 自力で集客・管理ができる自立型講師
- ポートフォリオ/LP 代わりとしてプラットフォームを使いたい層

### 生徒（受講者）側

- プラットフォーム上での決済なし（直接取引）
- 講師を「発見」する場として利用

---

## **3. 機能要件 (Functional Requirements)**

### **3.1 認証・ユーザー管理**

- **認証:** メールアドレス/パスワード (Supabase Auth)
- **ユーザー種別:** 講師のみ（MVP 段階では生徒アカウントは不要）

### **3.2 講師プロフィール**

講師の基本的な名刺情報。シンプルに保ち、コンテンツはスペースに集約する。

- **必須項目:**
  - 表示名
  - アバター画像
- **その他:**
  - 紹介コード（メンター間紹介用）

> **注意:** 外部リンク・自己紹介・カテゴリ/タグ等はスペース（3.3）に集約。プロフィールに記載するとスペース契約を回避できてしまうため、プロフィールには含めない。

### **3.3 スペース（講座ページ）**

メンターが講座ごとに作成する公開ページ。**月額/年額の契約が必要**。

- **基本情報:**
  - タイトル
  - サムネイル画像（1:1）
  - 説明文（Editor.js ブロックエディタ）
  - カテゴリ（コンボボックス選択）
  - タグ（複数選択可 + カスタムタグ入力対応）
  - スラッグ（公開URL用: `/s/:slug`）
- **外部リンク:**
  - Website URL
  - X (Twitter) URL
  - Instagram URL
  - 任意の外部リンク（ラベル + URL の複数追加）
- **公開/非公開:** サブスクリプション契約後に公開可能

### **3.4 スペース公開ページ**

- スペースごとの公開ページ（`/s/:slug`）
- 説明文、外部リンク、タグの表示
- SEO 対応

### **3.5 スペース一覧・検索**

- カテゴリフィルター
- タグ検索
- キーワード検索
- 新着順表示

### **3.6 メンターページ**

- メンターごとのページ（`/mentors/:slug`）
- メンターの基本情報（表示名、アバター）表示
- そのメンターが持つ公開スペースの一覧を表示

### **3.7 サブスクリプション決済**

- **Stripe Checkout:** 月額 1,000 円（税別）/ スペース、年額 10,000 円（税別）/ スペース
- **Stripe Customer Portal:** 解約・支払い方法変更
- **Webhook:** サブスク状態の同期

### **3.8 クーポン機能**

#### 運営者発行クーポン

- Stripe Coupon 機能を利用
- 割引率（%OFF）または固定金額オフ
- 有効期限設定可能

#### メンター招待クーポン

- 既存メンターが招待コードを発行
- 被招待者: **1 ヶ月無料**
- Stripe Promotion Code 機能を利用
- 招待履歴の確認機能

## **4. 技術スタック**

| レイヤー     | 技術                        |
| ------------ | --------------------------- |
| Runtime      | Deno 2.x                    |
| Frontend     | Fresh                       |
| API          | Hono                        |
| Backend/DB   | Supabase (PostgreSQL, Auth) |
| 決済         | Stripe                      |
| Hosting      | Deno Deploy                 |
| エディタ     | Editor.js                   |
| 画像切り抜き | Cropper.js                  |

---

## **5. 開発スコープ外 (Out of Scope for MVP)**

- **レビュー/評価機能:** Phase 2 で実装
- **本人確認/バッジ:** Phase 2 で実装
- **プラットフォーム内決済:** 直接取引のため不要
- **エスクロー機能:** 直接取引のため不要
- **メッセージ機能:** 外部ツールで代替
- **生徒アカウント:** 講師検索はログイン不要

---

## **6. 成功指標 (Success Metrics)**

| 指標                   | 目標                                 |
| ---------------------- | ------------------------------------ |
| 登録講師数（無料含む） | 100 人                               |
| 有料転換率             | 5%以上                               |
| 月商ブレークイーブン   | 200 スペース × 1,000 円 = 20 万円/月 |
| 講師紹介による新規登録 | 20%以上                              |

---

## **7. コールドスタート戦略**

1. **SNS での直接アプローチ**
   - 「ココナラ 手数料 高い」「Menta 手数料」等で検索
   - 不満を呟いている講師に個別アプローチ

2. **ポートフォリオ機能としての訴求**
   - 「実績ページ」「簡易 HP」として無料で使えることをアピール
   - 既存プラットフォームの実績を画像で掲載 OK

3. **ニッチ特化**
   - 最初は「エンジニアのメンタリング」等、特定領域に集中
   - その領域で「ここに来れば必ず見つかる」状態を作る

---

## **8. 法的考慮事項**

- **運営者免責:** 「取引の当事者ではない」ことを明記
- **直接取引リスク:** 自己責任の注意喚起を表示
- **特商法表記:** 講師側に自身の表記を促す啓蒙
- **18 歳未満利用:** 必要に応じて年齢確認を検討
