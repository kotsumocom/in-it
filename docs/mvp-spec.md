# **メンタープラットフォーム: MVP Specification**

## **1. MVP の目的 (Goal)**

**検証仮説:**
「手数料ゼロ・低掲載料（月額 1,000 円）のメンターマーケットプレイスは、高手数料に不満を持つ講師層を惹きつけ、持続可能なビジネスとして成立するか？」

**ビジネスモデル:**

- 月額掲載料: **1,000 円（税別）**
- 取引手数料: **0%**（直接取引 OK・外部リンク自由）
- 決済: Stripe Subscription

---

## **2. ターゲットユーザー**

### 講師（メンター）側

- 既存プラットフォーム（Menta, ココナラ, ストアカ等）の高手数料（20-30%）に不満を持つ層
- 自力で集客・管理ができる自立型講師
- ポートフォリオ/LP 代わりとしてプラットフォームを使いたい層

### 生徒（受講者）側

- プラットフォーム上での決済なし（直接取引）
- 講師を「発見」する場として利用

---

## **3. 機能要件 (Functional Requirements)**

### **3.1 認証・ユーザー管理**

- **認証:** メールアドレス/パスワード (Supabase Auth)
- **ユーザー種別:** 講師のみ（MVP 段階では生徒アカウントは不要）

### **3.2 講師プロフィール**

- **必須項目:**
  - 表示名
  - 自己紹介（Markdown 対応）
  - カテゴリ/タグ（プログラミング、語学、音楽等）
  - アイコン画像 URL
- **外部リンク:**
  - 予約ページ（Calendly, TimeRex 等）
  - SNS（Twitter/X, Instagram 等）
  - 自社サイト
  - Zoom/Google Meet リンク
  - 任意の外部 URL

### **3.3 講師公開ページ（簡易 LP）**

- 講師ごとの公開プロフィールページ
- 外部リンクボタン表示
- SEO 対応（講師名 + カテゴリでの検索流入を狙う）

### **3.4 講師一覧・検索**

- カテゴリフィルター
- キーワード検索（名前、自己紹介）
- 新着順/カテゴリ順表示

### **3.5 サブスクリプション決済**

- **Stripe Checkout:** 月額 1,000 円（税別）
- **Stripe Customer Portal:** 解約・支払い方法変更
- **Webhook:** サブスク状態の同期

### **3.6 クーポン機能**

#### 運営者発行クーポン

- Stripe Coupon 機能を利用
- 割引率（%OFF）または固定金額オフ
- 有効期限設定可能

#### メンター招待クーポン

- 既存メンターが招待コードを発行
- 被招待者: **1 ヶ月無料**
- Stripe Promotion Code 機能を利用
- 招待履歴の確認機能

---

## **4. 技術スタック**

| レイヤー   | 技術                        |
| ---------- | --------------------------- |
| Runtime    | Deno 2.x                    |
| Frontend   | Fresh                       |
| API        | Hono                        |
| Backend/DB | Supabase (PostgreSQL, Auth) |
| 決済       | Stripe                      |
| Hosting    | Deno Deploy                 |

---

## **5. 開発スコープ外 (Out of Scope for MVP)**

- **レビュー/評価機能:** Phase 2 で実装
- **本人確認/バッジ:** Phase 2 で実装
- **プラットフォーム内決済:** 直接取引のため不要
- **エスクロー機能:** 直接取引のため不要
- **メッセージ機能:** 外部ツールで代替
- **生徒アカウント:** 講師検索はログイン不要

---

## **6. 成功指標 (Success Metrics)**

| 指標                   | 目標                           |
| ---------------------- | ------------------------------ |
| 登録講師数（無料含む） | 100 人                         |
| 有料転換率             | 5%以上                         |
| 月商ブレークイーブン   | 200 人 × 1,000 円 = 20 万円/月 |
| 講師紹介による新規登録 | 20%以上                        |

---

## **7. コールドスタート戦略**

1. **SNS での直接アプローチ**

   - 「ココナラ 手数料 高い」「Menta 手数料」等で検索
   - 不満を呟いている講師に個別アプローチ

2. **ポートフォリオ機能としての訴求**

   - 「実績ページ」「簡易 HP」として無料で使えることをアピール
   - 既存プラットフォームの実績を画像で掲載 OK

3. **ニッチ特化**
   - 最初は「エンジニアのメンタリング」等、特定領域に集中
   - その領域で「ここに来れば必ず見つかる」状態を作る

---

## **8. 法的考慮事項**

- **運営者免責:** 「取引の当事者ではない」ことを明記
- **直接取引リスク:** 自己責任の注意喚起を表示
- **特商法表記:** 講師側に自身の表記を促す啓蒙
- **18 歳未満利用:** 必要に応じて年齢確認を検討
