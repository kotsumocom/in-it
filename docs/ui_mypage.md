# マイページ UI仕様書

## 概要
講師のダッシュボード。シングルカラム。

---

## ページ構成

### 1. ヘッダー
- ロゴ / 「プロフィールを見る」 / ログアウト

### 2. サブスク状態
| 状態 | 表示 |
|------|------|
| active | 緑バッジ + 次回請求日 |
| past_due | 黄バッジ + 警告 |
| canceled | グレー + 再開ボタン |

→ 「プランを管理」でStripe Portal

### 3. プロフィール管理
- 「編集する」→ **別ページ**へ遷移

### 4. 本人確認
- 未確認: 「本人確認を始める」ボタン
- 確認済み: バッジ表示

### 5. 招待機能

```
招待コード: ABC123 [コピー]

✨ 招待特典
・招待された人: 1ヶ月無料
・招待した人: 1ヶ月無料（次回請求時）

招待履歴: 3人
```

---

## 招待クーポン セキュリティ設計

### Stripe設定
| 項目 | 設定 |
|------|------|
| 割引 | 100% OFF（1ヶ月） |
| 適用対象 | **初回注文のみ** |
| 顧客制限 | 1メールにつき1回 |
| コード形式 | ユーザー固有コード |

### 悪用対策
- **初回限定**: 既存顧客は使用不可
- **1アカウント1回**: 同一メールで再登録不可（Stripe側で管理）
- **招待者報酬の条件**: 被招待者が**有料決済完了後**に付与
- **監視**: 異常な招待パターン検知（大量招待等）

### リスクと対策
| リスク | 対策 |
|--------|------|
| コード流出 | 初回限定+1回制限で影響限定 |
| 自己招待 | 同一IP/決済情報で検知可能 |
| 大量アカウント作成 | メール確認必須+手動審査オプション |
