OGP Background — 1200×630

HTML要素を画像として保存する方法：

方法1: DevTools
要素を右クリック → 検証 → Elements パネルで #ogp-canvas を右クリック → Capture node screenshot

方法2: html2canvas (JS ライブラリ)
<script src="https://html2canvas.hertzen.com/dist/html2canvas.min.js"></script>
html2canvas(document.getElementById('ogp-canvas')).then(c => { a = document.createElement('a'); a.download = 'ogp-bg.png'; a.href = c.toDataURL(); a.click(); });

方法3: ブラウザ拡張
GoFullPage, FireShot 等で要素キャプチャ
